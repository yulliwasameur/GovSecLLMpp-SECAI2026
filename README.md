# GovSecLLM++ SECAI 2026 Artifact Package

This repository hosts the reproducibility artifact package for:

**GovSecLLM++: A Compliance-Aware Benchmark for Security Testing and Governance Evidence in LLM-Based Applications**

The current release contains the experimental artifact package without the final paper source/PDF. The package includes:

- AdaptiveGovSec-120 outputs;
- corrected F2 rerun outputs;
- mini multi-model benchmark outputs;
- strict rescoring files;
- human-validation summary workbook;
- scoring utilities;
- data card;
- reproducibility notes;
- checksums and manifest.

The paper source and final PDF may be added in a later release.
